# FIGHT CVMP Landing Page

A cinematic, high-impact landing page for FIGHT CVMP - a performance platform built for high-agency individuals.

## Description

Structure Your Mind. Build Your Body. Expand Your Capacity.

For fighters, athletes, entrepreneurs, and high-agency women & men who refuse regression.

## Features

- Dark, cinematic theme with dramatic imagery
- Fully responsive design (mobile & desktop)
- Waitlist form with email capture
- Three core pillars:
  - **STATE CONTROL** - Regulate your nervous system, master activation & composure
  - **STRUCTURE** - Install protocols that hold, reinforce habits and standards
  - **CAPACITY EXPANSION** - Train baseline-raising behavior, become resilient under pressure

## Tech Stack

- Pure HTML5
- Embedded CSS3
- Vanilla JavaScript
- Google Fonts (Montserrat)
- Unsplash imagery
